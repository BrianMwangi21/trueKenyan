# truekenyan
A blog for the people, by the people

---
### How to view
1. Install Metamask extention on your browser https://metamask.io/ and create an account
2. Once done, change your network from Ethereum Main Network to Kovan Test Network
3. Head over to https://gitter.im/kovan-testnet/faucet and request some ether by pasting your address. Note : **Ether in testnets is not real and holds no value**
4. Head over to https://truekenyan.herokuapp.com/ and enjoy!
